<br />
Are you looking for lightweight Ajax Contact Form. This plugin mainly design for Contact Form, also this plugin will help you to send mail using ajax from your wordpress site. You can design your own contact form from the admin. You can add and remove any type of field in the form on fly.
There will be a email list waiting for your review. There are several options for the contact that you can update later.
You just need to put shortcode [wpajaxcf] in any page/post. There is also custom css area in the admin.

*	ajax with contact form

*	simple to use and update
*	custom css

== Installation ==

Manual installation:

1. Upload the `wp-ajax-contact-form` folder to the `/wp-content/plugins/` directory

Installation using "Add New Plugin"

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
2. Search for ' WP Ajax Contact Form'
3. Click the 'Install' button to open the plugin's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the plugin through the 'Plugins' menu in WordPress
2. From your Admin UI (Dashboard), use the menu to select "wpAjaxcf Options->Settings" 
3. Configure settings, and save
